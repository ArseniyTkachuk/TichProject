
# Сайт з тестами

Цей репозиторій містить **фронтенд та бекенд** для веб-додатку з тестами. Користувачі можуть переглядати, проходити, створювати та видаляти тести.

---

## Структура репозиторію

```

/frontend   # Vue 3 + Vite
/backend    # Node.js / Express 

````

---

## Фронтенд

### Технології

- Vue 3 + Vite  
- Vue Router для навігації  
- Axios для HTTP-запитів  
- CSS (Flexbox, градієнти, модальні вікна)  

### Встановлення та запуск

1. Перейти в каталог фронтенду:
```bash
cd frontend
````

2. Встановити залежності:

```bash
npm install
```

3. Створити `.env` файл з адресою бекенду:

```env
VITE_BACK_URL=http://localhost:2222
```

4. Запустити локальний сервер:

```bash
npm run dev
```

5. Відкрити у браузері: [http://localhost:5173](http://localhost:5173)

---

## Бекенд

### Технології

* Node.js + Express 
* MongoDB/PostgreSQL
* Роутінг для роботи з тестами

### Встановлення та запуск

1. Перейти в каталог бекенду:

```bash
cd backend
```

2. Встановити залежності:

```bash
npm install
```

3. Створити `.env` файл з налаштуваннями бази даних:

```env
PORT=2222
```

4. Запустити сервер:

```bash
npm run dev
```

---

## Основні функції додатку

* Перегляд списку тестів
* Проходження тестів з різними типами завдань:

  * Один варіант відповіді
  * Кілька варіантів відповіді
  * Введення тексту
  * Парні завдання
* Поділитися тестом (код або посилання)
* Видалення тесту
* Статистика проходжень
