# Сайт з тестами

Це веб-додаток для створення, проходження та управління тестами. Користувачі можуть переглядати тести, проходити їх, ділитися тестами та видаляти власні тести.

---

## Основні функції

- Перегляд списку тестів
- Пройти тест (різні типи завдань: вибір одного або кількох варіантів, введення тексту, парні завдання)
- Поділитися тестом за кодом або посиланням
- Видалення тесту
- Статистика проходжень тесту

---

## Технології

- **Фронтенд:** Vue 3, Vite
- **Стилі:** CSS (Flexbox, градієнти, модальні вікна)
- **HTTP-запити:** Axios
- **Роутінг:** Vue Router
- **Середовище:** Node.js, Vite
- **Бекенд:** будь-який REST API 

---

## Встановлення та запуск

1. Клонувати репозиторій:
```bash
git clone <url-репозиторію>
````

2. Встановити залежності:

```bash
npm install
```

3. Створити `.env` файл у корені проекту та додати URL бекенду:

```env
VITE_BACK_URL=http://localhost:3000
```

4. Запустити локальний сервер:

```bash
npm run dev
```

5. Відкрити у браузері: [http://localhost:5173](http://localhost:5173)

---

## Структура проекту

```
/src
  /components
    /loockTest
      answers.vue
      enters.vue
      pairs.vue
  /views
    TestView.vue
  App.vue
  main.js
```

* `components/loockTest` — компоненти для різних типів завдань
* `views/TestView.vue` — сторінка перегляду тесту
* `App.vue` та `main.js` — основний код Vue-додатку

---

## Використання

* Кнопка **Поділитися** відкриває модальне вікно з кодом тесту та посиланням
* Кнопка **Видалити** відкриває підтверджувальне модальне вікно та видаляє тест після підтвердження
* Статистика тесту відображає кількість проходжень та завдань
* Різні типи завдань обробляються через компоненти `answers`, `enters` та `pairs`
